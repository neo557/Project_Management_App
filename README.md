# 企画管理アプリ（Project Manager）

軽量で高速な 企画管理・ファイル管理ツール。
個人制作・同人開発・研究管理などに向けて設計された、
PyQt5 製のデスクトップアプリケーションです。

## Features
- 企画管理（タイトル / 開始日 / 責任者 / 状態 / メモ）

- カテゴリ分類（一般 / 開発 / 研究 / その他）

- 企画ごとに複数ファイルを紐付け可能

- UUID＋相対パス保存で安全性が高い

- 保存先フォルダをユーザーが自由に設定可能

- ファイルの上書き事故ゼロ

- タスクトレイ常駐

- PyInstaller で単体 EXE 化可能（コンソールなし）

## Folder Structure
初回起動時に保存先フォルダを選択すると、
以下のカテゴリフォルダが自動生成されます：

保存先/
│ 一般/
│ 開発/
│ 研究/
│ その他/

企画に紐付けられたファイルは、
UUID + 元ファイル名 の形式で保存されます：

開発/
  3f2a9c1e-xxxx-xxxx-xxxx-xxxx_myfile.png
## Safety Design
このアプリは、ユーザーのファイルを安全に扱うために
以下の設計を採用しています。

✔ UUID 保存
同名ファイルを登録しても上書きされません。

✔ 相対パス管理
アプリを別フォルダに移動してもファイル参照が壊れません。

✔ 削除時は例外処理付き
破損ファイルがあってもアプリが落ちません。

✔ 保存先フォルダはユーザーが選択
初回起動時に自由に設定できます。

## Usage
1. 初回起動
保存先フォルダを選択します。
ここにカテゴリフォルダが自動生成されます。

2. 企画追加
画面下部のフォームから企画を登録できます。

3. ファイル追加
企画ごとに複数ファイルを紐付けできます。

4. ファイルを開く
ファイル一覧のリンクをクリックすると開きます。

5. タスクトレイ常駐
アプリを閉じてもタスクトレイに常駐します。

## Dependencies
- Python 3.10+

- PyQt5

- sqlite3

- PyInstaller（ビルド時）
